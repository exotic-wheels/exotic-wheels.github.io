# exotic-wheels

> Pre-built wheels for platforms PyPI forgot.

## Install

```pip install cryptography -f exotic-wheels.github.io```

## About

This index serves pre-built wheels for exotic platforms (musl ARM, 32-bit, legacy glibc, PyPy, GraalPy, free-threaded CPython) that mainstream PyPI doesn't cover.

Wheels are built, signed, and released at **[1minds3t/exotic-wheels](https://github.com/1minds3t/exotic-wheels)**.
